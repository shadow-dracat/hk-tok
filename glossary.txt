NOTES:
- for tokiponizations I usually go for the "preserve syllable count" way.
- essence-related things use "pilin", infection can use "jaki". Deamers and the word "nightmare" can still use "lape", fools are "nasa"
- default headnouns are "pipi". Hunter is "monsuta" because it has those vibes to me.
- I try to avoid pi nesting as mush as possible. The main place where this would happen is achievements. This influences the naming process.
- endearment terms can be "suwi", "pipi suwi" or "pipi suwi mi" etc.
- exclamations like "oh goodness", "oh dear" etc. are "sewi o".
- I don't prolong "n" for "mmmm"/"hhmmmmm", but I can prolong "a" and "mu". This convention can be changed, and ultimately, this doesn't matter as much.
